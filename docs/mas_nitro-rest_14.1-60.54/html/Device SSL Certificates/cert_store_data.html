cert_store_data

Configuration for SSL certificate file store on NetScaler Console resource.

Read/write properties

file_name<String>

Certificate file name..
Minimum length = 1
Maximum length = 128

file_data<String>

Base64 encoded certificate file contents..
Maximum length = 50000

id<String>

Id is system generated key for all certificate store data entries. .

referenced_ic_cert_count<Integer>

Number of certificates(cert_store) referring this intermediate certificate across instances.

Read only properties

fingerprint<String>

Fingerprint of certificate file.

certkeypair_name<String>

Cert Key Pair Name.

is_installed<Boolean>

Indicates if a cert is installed in an ADC,when display_name and action parameter are set in filter query.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
