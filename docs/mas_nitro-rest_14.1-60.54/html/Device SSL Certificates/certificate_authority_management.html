certificate_authority_management

Configuration for Third party CA provider resource.

Read/write properties

policy_folder_list<String>

Comma separated list of policies available under policy_folder. Applicable only if the CA provider is Venafi..

refresh_token<String>

Before the access token expires, application uses the refresh token to get a new set of tokens.

name<String>

Name to identify the third party CA management server..

renew_before_days<Integer>

Number of days before which certificates managed by third party CA should be renewed by NetScaler Console.Should be set only if is_renew is true..

agent_id<String>

Id of the agent which connects to third party CA management server.

is_renew<Boolean>

Whether Auto renewal of certificates managed by third party CA should be done by NetScaler Console..

passphrase<String>

password with which the cert/key file will be encrypted when downloading from third party CA management server.

device_folder<String>

Device Folder.Applicable only if the CA provider is Venafi.

api_endpoint<String>

WebSDK endpoint for the configured third party CA management server..

access_token<String>

Access token is passed in the header of every API call.

ca_provider<String>

CA provider.Possible values are: Venafi.

client_id<String>

Client id required to make API calls to third party CA management server.

id<String>

Id is system generated key for all third party CA management resource..

policy_folder<String>

Policy Folder.Applicable only if the CA provider is Venafi.

is_deploy<Boolean>

Whether Auto deployment of certificates managed by third party CA should be done by NetScaler Console.Should be set only if is_renew is true..
