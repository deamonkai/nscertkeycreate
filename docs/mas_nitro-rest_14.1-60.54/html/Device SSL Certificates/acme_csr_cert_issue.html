acme_csr_cert_issue

Configuration for ACME CSR cert issue configuration resource.

Read/write properties

dns_provider_id<String>

ID of the DNS provider configuration used for DNS challenges..

file_data<String>

Base64 encoded CSR contents..
Maximum length = 50000

key_data<String>

Base64 encoded key file contents..
Maximum length = 16384

name<String>

Name of the new certificate issue configuration..
Maximum length = 128

key_csr_available_on_server<Boolean>

Indicates whether the CSR and key have been already uploaded to the server under the default location. The same will be referred for certificate creation..

acme_ca_config_id<String>

ID of the CA config that is used to issue the certificate.

key_file<String>

Key file name.
Minimum length = 1
Maximum length = 128

store_type<String>

Type of store where the certificate will be stored. It can be 'cert_store' or 'zero_touch'..

is_auto_renewal<Boolean>

Set to true, if the certificate should be automatically renewed before expiry..

activity_id<String>

Activity ID of the certificate issue operation..

domain<String>

Domain name from the subject of the certificate.

key_length<String>

Key length used for the ACME CA configuration. Values: 1.2048, 2.4096, 3.ec-256, 4.ec-384..

file_name<String>

Certificate file name..
Minimum length = 1
Maximum length = 128
