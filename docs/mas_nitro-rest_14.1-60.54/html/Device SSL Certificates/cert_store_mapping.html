cert_store_mapping

Configuration for Mapping between the certificates in store and the entities to which they are bound resource.

Read/write properties

entity_id<String>

Id of the application or module to which the cert is bound.
Minimum length = 1
Maximum length = 128

installed_cert_data_ids<String>

Comma separated list of cert store data ids installed.

cert_id<String>

Id of the certificate file in cert store table.
Minimum length = 1
Maximum length = 128

entity_name<String>

Name of the application or module to which the cert is bound.
Minimum length = 1
Maximum length = 128

instance_host_name<String>

Host name of the instance in the managed_device table, in which the cert is installed.
Minimum length = 1
Maximum length = 128

entity_type<String>

Type of the application or module to which the cert is bound.
Minimum length = 1
Maximum length = 128

instance_id<String>

Id of the instance in the managed_device table, in which the cert is installed.
Minimum length = 1
Maximum length = 128

instance_display_name<String>

Display name of the instance in the managed_device table, in which the cert is installed.
Minimum length = 1
Maximum length = 128

id<String>

Id is system generated key for all certificate story entries. .

instance_ip<String>

IP Address of the instance in the managed_device table, in which the cert is installed.
Minimum length = 1
Maximum length = 128

is_instances<Boolean>

To retrieve only the instances associated with the provided cert_id.

is_entities<Boolean>

To get only the entities associated with the provided cert_id.
