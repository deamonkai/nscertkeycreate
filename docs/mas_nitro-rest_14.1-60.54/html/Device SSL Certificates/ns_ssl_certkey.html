ns_ssl_certkey

Configuration for SSL certificate on NetScaler resource.

Read/write properties

certkey_status<String>

Status(ACTIVE/INACTIVE) of certkey based on its traffic..

ssl_key<String>

Key.
Minimum length = 1
Maximum length = 128

file_location_path<String>

File Location on Client for download.
Minimum length = 1

id<String>

Id is system generated key for all ssl cert-keys entries. For download operation "id" must be provided in the format <adc_ip_address>_<certkeypair_name>.tgz.

certificate_source<String>

This will have the source(LOCAL/REMOTE) of the certificate. Zero touch certificates will have the source as REMOTE..

cert_format<String>

Certificate Format.
Maximum length = 64

certkeypair_name<String>

Cert Key Pair Name.
Minimum length = 1
Maximum length = 128

managed_by<String>

Indicates if the certificate is managed by a third party certificate management provider..

linkcertkeyname<String>

For certificates having an intermediate certificate, it will provide the linked certificate key pair name.

certificate_data<String>

Certificate Data.
Maximum length = 50000

ssl_certificate<String>

Certificate.
Minimum length = 1
Maximum length = 128

no_domain_check<Boolean>

Specify this option to override the check for matching domain names during certificate update operation.

ns_ip_address<String>

List of NetScaler IP Address.
Minimum length = 1
Maximum length = 64

certificate_dn<String>

For certificates created by Venafi,this property indicates the certificate distinguished name..

key_data<String>

Key Data.
Maximum length = 16384

deleteCertKeyFilesOnRemoval<String>

Argument to remove certkey files from NS along with certificate configuration based on the option selected by the user.

ns_ip_address_arr<String[]>

List of NetScaler IP Address.
Minimum length = 1
Maximum length = 64

file_name<String>

File Name.
Minimum length = 1
Maximum length = 256

issuer_cn<String>

CN of issuer.

save_config<Boolean>

true, if save config is required.

snicert<Boolean>

Indicates server certificate for SNI. Used in cert binding operation..

certchainbinding<String[]>

Certificate Chain binding..

certchain_data<cert_store_data[]>

An array of Base64 encoded certificate chain contents..

entity_name<String>

entity name.

password<String>

The pass-phrase that was used to encrypt the private-key..
Maximum length = 128

cert_ids<String[]>

Array of SSL certificate IDs.

key_file_name<String>

Key File Name.

days_to_expiry_str<String>

Days to expire in years and days.

certkeypair_arr<String[]>

List of certkeypair_name.Used in cert binding operation..

source_ipaddress<String>

NetScaler IP of the certificate that needs to installed in another instance.

certificate_file_name<String>

Certificate File Name.

app_name<String>

Application name.Used for tracking task logs created for a specific application.

entity_binding_arr<ns_sslcertkey_binding[]>

.

certkeychain<certkey_chain_link[]>

Array of linked certificates details.

source_certificate<String>

CertKeyPair Name of the certificate that needs to installed in another instance.

Read only properties

no_of_bound_entities<Integer>

no_of_bound_entities.

partition_name<String>

Name of Admin Partition. Blank means Default Partition.

subjaltname<String>

Comma separated Subject Alternative Names.

days_to_expiry<Integer>

Days before SSL certificate expires.

valid_to<String>

Valid To.

signature_algorithm<String>

Signature Algorithm.

public_key_size<Integer>

Public Key Size.

display_name<String>

Display Name of the device.

status<String>

Tells whether the certificate is still valid or not.

hostname<String>

Host Name of the device.

public_key_algorithm<String>

Public Key Algorithm.

version<Integer>

Version.

device_name<String>

Name of the device.

valid_from<String>

Valid From.

issuer<String>

Issuer.

poll_time<Integer>

Last Polling Time.

serial_number<String>

Serial Number.

subject<String>

Subject.

cert_store_id<String>

Cert Store Id .

activity_id<String>

Activity Id used to track the progress of certificate operations..

csr<String>

Certificate Signing Request.

domain<String>

Domain name for certificate issuer.

datacenter_id<String>

Datacenter Id .
