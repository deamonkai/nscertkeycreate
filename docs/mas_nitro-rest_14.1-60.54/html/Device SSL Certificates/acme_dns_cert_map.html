acme_dns_cert_map

Configuration for ACME DNS Cert Domain mapping configuration resource.

Read/write properties

domain<String>

Domain name from the subject of the certificate.

issued_by_org<String>

Issued Organization from the subject of the certificate.

name<String>

Name of the cert.

id<String>

Id is system generated key for DNS mapping configuration..

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
