acme_ca_config

Configuration for ACME CA Configuration resource.

Read/write properties

agent_id<String>

ID of the agent that is used to connect to ACME CA server.

email_id<String>

Email Address used for ACME CA configuration.

acme_ca_server_type<String>

ACME CA Server Type. Values: 1.LETS_ENCRYPT, 2. DIGICERT.
Maximum length = 128

is_enabled<Boolean>

Set to true, if the CA is enabled for certificate issue/renew.

acme_server_config_map<acme_access_parameters[]>

Array of key and value for acme server configuration.

name<String>

Name of the CA Configuration. This is a user defined name for the ACME CA configuration..
Maximum length = 128

status<String>

DRAFT/COMPLETE. Status of the ACME CA configuration. DRAFT means the configuration is not yet complete and cannot be used for certificate issue/renew. COMPLETE means the configuration is complete and can be used for certificate issue/renew..
Maximum length = 24

challenge_type<String>

Challenge type used for ACME CA configuration. Values: 1.HTTP-01, 2.DNS-01, 3.TLS-ALPN-01.

created_time<Double>

Time when the ACME CA configuration was created in seconds.

updated_time<Double>

Time when the ACME CA configuration was last updated in seconds.

key_length<String>

Key length used for the ACME CA configuration. Values: 1.2048, 2.4096, 3.ec-256, 4.ec-384..

renew_interval<Integer>

Expiry day for renewing the certificate.
Minimum value = 1
Maximum value =

id<String>

Id is system generated key for all ACME CA entries..

supported_ca_list<Boolean>

Set to true, if only password needs to be updated during update operation.

is_incomplete_deployment<Boolean>

Set to true, if the CA configuration is incomplete and cannot be used for certificate issue/renew.

get_by_draft_status<Boolean>

Set to true, if only DRAFT status ACME CA configurations need to be fetched.

Read only properties

agent_type<String>

Type of agent. Values: 1.External_Agent, 2. Inbuilt_Agent.
