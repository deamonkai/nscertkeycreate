ns_config_audit_settings

Configuration for Config Audit Setting resource.

Read/write properties

audit_on_trap_receive<Boolean>

Run config audit for an NetScaler instance after config change trap is received.

time_after_receive<Integer>

Time (in minutes) after which Config Audit is run for a particular IP after config change trap is received.

is_template_diff_bidirectional<Boolean>

Indicates whether template Vs running diff is bi-directional.

id<String>

Id is system generated key..
