vpn_user_info

Configuration for Af report for VPN User resource.

Read/write properties

id<String>

Id is ICA User Name.
Minimum length = 1
Maximum length = 64

session_count<Double>

Number of sessions launched by user.

bandwidth<Double>

Avg Bandwidth..

rpt_sample_time<Double>

Report Sample time..

active_session<Double>

Number of active sessions launched by user.

name<String>

Name of ICA User.
Minimum length = 1
Maximum length = 128

session_used<Double>

Number of sessions used by user.

logon_duration<Double>

Logon duration of user in seconds.

total_bytes<Double>

Total bytes in sampled timeframe used by all users..

__count<Double>

count..
