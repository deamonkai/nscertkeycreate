protection_info

Configuration for Mapping Unified Appsec profile with Protection Info resource.

Read/write properties

id<String>

Id is system generated key for protection.

protection_id<Integer>

ID to validate the Protection.

type<String>

Protection category.

parent_name<String>

Resource name of parent resource.

parent_id<String>

Id of the parent resource.

protection_name<String>

Protection name.

enable_keys<String>

Protection rules enable keys.

config<String>

Protection rules.
