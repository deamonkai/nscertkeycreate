adm_learn_profile

Configuration for NetScaler Console WAF Profile to manage learning groups resource.

Read/write properties

no_of_learned_rules<Integer>

Number of WAF Profile names configured.

waf_profile_devices_info<waf_profile_devices[]>

Mapping NetScaler IPs to WAF Profiles.

waf_security_settings_info<waf_security_settings[]>

WAF Security Checks Settings info.

no_of_deployed_rules<Integer>

Number of WAF Profile names configured.

slack_profile<String>

Slack Profile.

servicenow_profile<String>

ServiceNow Profile Name.

adm_learn_profile_name<String>

Profile Name used for naming a learning group.
Maximum length = 128

enabled<Boolean>

enable learning on this profile.

mail_profile<String>

Mail Profile.

learning_mode<Integer>

WAF Learning modes can be WAF Profile name based (0), Application name based (1), Rule removal (2), Hybrid (3) .

sms_profile<String>

SMS Profile.

no_of_adc<Integer>

Number of ADCs.

no_of_waf<Integer>

Number of WAF Profile names configured.

Read only properties

id<String>

Id is system generated key..
