http_header

Configuration for Enable/Disable HTTP header info for user_agent, http_req_method, http_resp_status, operating_system as 1st level tree nodes. resource.

Read/write properties

http_resp_status<Boolean>

Enable/Disable Http Resp Status report.

user_agent<Boolean>

Enable/Disable user agent report.

__count<Double>

Count..

ic_nostore_reason<Boolean>

Enable/Disable Cache No Store Reason report.

http_req_method<Boolean>

Enable/Disable http req method report.

domain<Boolean>

Enable/Disbale Domain report .

operating_system<Boolean>

Enable/Disbale Operating System report .

http_media_type<Boolean>

Enable/Disable Http Media Type report.

http_content_type<Boolean>

Enable/Disable Http Content Type report.

NOTE: In request payload, Mandatory parameters are marked red and bold.

Possible values = EXIT, CONTINUE
