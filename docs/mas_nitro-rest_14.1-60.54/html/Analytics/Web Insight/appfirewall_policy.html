appfirewall_policy

Configuration for Appfirewall Policy Hit Counters Report resource.

Read/write properties

ns_ip_address<String>

IP address of the NetScaler the policy is bound to..

rule<String>

Rule name configured on ADC..

schema_type<String>

One of the supported schema types..

pcp_hits<Integer>

PCP hits of the counter.

policy_name<String>

Policy name configured on ADC..

__count<Double>

count..

network_function<String>

network_function name configured on ADC..

rpt_sample_time<Double>

Report Sample time..

application_name<String>

application_name.
