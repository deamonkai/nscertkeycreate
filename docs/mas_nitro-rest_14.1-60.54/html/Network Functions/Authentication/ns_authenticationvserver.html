ns_authenticationvserver

Configuration for NetScaler Auth vserver resource.

Read/write properties

analytics_managed<String>

whether the analytics is auto_enabled/manual_enabled/manual_disabled....

unified_appsec_profile_name<String>

unified appsec profile bound to this vserver.

metrics_enabled<Boolean>

Flag to indicate Metrics Enablement for the given vserver.

monitor_mode_toggle<Boolean>

Unified Appsec Monitor mode toggle.

export_list<String>

Export List present for the given LB VServer.

secure_vsvr_state<Integer>

VServer state wheather it is secured or not.

ns_ip_address<String>

NetScaler IP Address.
Minimum length = 1
Maximum length = 64

name<String>

Vserver Name.
Minimum length = 1
Maximum length = 100

appsec_profile_name<String>

appsec profile bound to this vserver.

throughput<Double>

throughput value.

user_managed<Boolean>

User Managed Idenfication.

export_option<String>

Export Options present for the given LB VServer.

es4nslog<String>

ESNS enable.

metrics_option_list<String>

Metric usecases user wants to enable.

configpack_id<String>

Stylebooks config pack ID.

appflow_policy_rule<String>

AppFlow policy rule present.

appflowlog_effective<String>

Appflowlog effective (validating AppFlow config) enabled or disabled.

unified_appsec_protection_count<Integer>

unified appsec protection count to this vserver.

instance_license<String>

Feature License for NetScaler Instance, needs to be set while provisioning (standard, enterprise, platinum).
Minimum length = 1
Maximum length = 64

appflowlog<String>

Appflowlog enabled or disabled.

transaction_log_effective<String>

Transaction Log enable/disable.

vsvr_ip_address<String>

Vserver IP Address.
Minimum length = 1
Maximum length = 64

hostname<String>

Hostname of the managed device.
Minimum length = 1
Maximum length = 256

instance_id<String>

Netscaler ID.

httpxforwardedfor<String>

X-Forwarded-For Header Export status.

icalog<String>

ICA log.

agent_list<String>

Agent List, on which traffic will flow.

id<String>

Id is system generated key.

transport_mode<String>

Transport type - IPFix or LogStream.

client_header<String>

Custom Client Header to Export.

csvip_id<String>

CSVIP ID.

save_config<Boolean>

Save configuration after enable/disable.

is_throughput_req<Boolean>

Throghput required.

is_bot_policy_bound<Boolean>

Is Bot Policy bound to vserver..

csvip_name<String>

CSVIP Name.

app_security<String>

App Security to know whether bot and appfw policies are bound..

svc_grp_name<String>

Service Group Name.

is_appfw_policy_bound<Boolean>

Is Appfw Policy bound to vserver..

svc_id<String>

Service ID.

svc_grp_id<String>

Service Group ID.

svc_name<String>

Service Name.

Read only properties

device_type<String>

Device Type : vpx, cpx, blx, cpx-sidecar.

display_name<String>

Display Name of Vserver.

comment<String>

Vserver comment.

targetlbvserver<String>

Default Target LBVserver.

state<String>

State of VServer.

license_type<Integer>

License Type : 0 - not licensed, 1 - manual, 2 - vpx, 3 - cpx, 4 - auto, 5 - cpx-sidecar/free.

cachetype<String>

Cache Type.

hits<String>

Traffic Domain.
