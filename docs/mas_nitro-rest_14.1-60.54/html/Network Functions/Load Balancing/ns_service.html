ns_service

Configuration for NetScaler Services Information resource.

Read/write properties

ns_ip_address<String>

NetScaler IP Address.
Minimum length = 1
Maximum length = 64

svc_ip_address<String>

Service IP Address.
Minimum length = 1
Maximum length = 64

svc_name<String>

Service Name.
Minimum length = 1
Maximum length = 100

id<String>

Id is system generated key.

publicip<String>

Public IP Address.
Minimum length = 1
Maximum length = 64

hostname<String>

Hostname of the managed device.
Minimum length = 1
Maximum length = 256

appflowlog<String>

AppFlow is enabled at service of LBVServer.

graceful<Boolean>

Graceful shutdown of Service..

vsvr_name<String>

Vserver Name.
Minimum length = 1
Maximum length = 100

lb_id<String>

LB ID.

svr_id<String>

Server ID.

delay<Double>

Time, in seconds, allocated to the NetScaler appliance for a graceful shutdown of the service.

save_config<Boolean>

Save configuration after enable/disable.

Read only properties

display_name<String>

Display Name.

partition_name<String>

Partition Name.

svc_type<String>

Service Protocol (HTTP|FTP).

svc_state<String>

Service State.

site_name<String>

Site Name.

svc_port<Integer>

Service Port.

poll_time<Integer>

Last Polling Time.

svr_name<String>

Server Name associated with service.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
