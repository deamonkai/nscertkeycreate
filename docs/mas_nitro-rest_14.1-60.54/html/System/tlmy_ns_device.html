tlmy_ns_device

Configuration for The table maintained by telemetry insights module which has subset of managed_device table columns. resource.

Read/write properties

device_state<String>

State of the device - Up/Down.

last_seen<Double>

The time specified as epoch in seconds when the device was last seen by tlmy insights.

device_version<String>

Version of the device.

md_id<String>

Device Id from tenant's managed_device table.

device_family<String>

Family of the device - NetScaler/SVM etc..

device_ip<String>

IP address of the device.

agent_ip<String>

IP address of the agent managing the device.

org_id<String>

Org id to which the device belongs.

managed_by<String>

Type of the agent managing the device.

first_discovered<Double>

The time specified as epoch in seconds when the device was 1st seen by tlmy insights.

agent_state<String>

State of the agent managing the device.

agent_version<String>

Version of the agent managing the device.

id<String>

.

cc_id<String>

CCID to which the device belongs; TODO: add UNIQUE (device_id, ccid) constraint through code/SQL.
