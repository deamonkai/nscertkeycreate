feature_migration_status

Configuration for Feature Migration Status resource.

Read/write properties

message<String>

Details about feature migration.

tables_mig_success<Double>

Number of tables successful during migration to new database.

feature_status<String>

Feature Status: (pending, success, in-progress, failed).

tables_mig_pending<Double>

Number of tables pending for database migration.

tables_mig_failed<Double>

Number of tables failed during migration to new database.

total_tables<Double>

Total Tables.

feature_name<String>

Feature name.
