servicenow_profile

Configuration for Servicenow details. resource.

Read/write properties

url<String>

URL for ServiceNow Endpoint.

domain<String>

Domain for ServiceNow Endpoint.

tenant_id<String>

Tenant Id of the servicenow profile.

is_validated<Boolean>

true if this profile has been validated and can be used to raise the ticket.

profile_name<String>

Profile name through which servicenow messages will be sent..
Minimum length = 1
Maximum length = 128

id<String>

Id is system generated key for all the sent servicenow messages..

is_enabled<Boolean>

true if this profile has been enabled.
