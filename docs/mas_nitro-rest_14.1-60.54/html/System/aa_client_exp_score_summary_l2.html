aa_client_exp_score_summary_l2

Configuration for Client Experience score summary table for Level 2 resource.

Read/write properties

ip_address<String>

IP Address.
Maximum length = 512

client_rtt_affected_clients<Integer>

Number of clients affected by client_rtt issue..

total_clients<Integer>

Total clients per type..

load_time_affected_clients<Integer>

Number of clients affected by load_time issue..

render_time_affected_clients<Integer>

Number of clients affected by render_time issue..

ctnsappname<String>

AppName.
Maximum length = 255

ssl_failures_affected_clients<Integer>

Number of clients affected by ssl_failures issue..

client_type<String>

Type of clients - [ Good, Fair, Bad ] based on their experience score.
Maximum length = 255

rpt_sample_time<Double>

Report Sample time..

client_failures_affected_clients<Integer>

Number of clients affected by client_failures issue..
