tacacs_server

Configuration for TACACS Server configuration resource.

Read/write properties

name<String>

Name of TACACS server.
Minimum length = 1
Maximum length = 128

port<Integer>

port number of TACACS server.
Minimum value = 1
Maximum value =

group_attr_name<String>

The Attribute name for group extraction from the ACS server. If not passed, then groups will not be extracted. No other harm.
Maximum length = 64

id<String>

Id is system generated key for all the TACACS servers.

tacacs_key<String>

Key shared between the TACACS+ server and clients.
Minimum length = 1
Maximum length = 64

ip_address<String>

IP Address of TACACS server.
Minimum length = 1
Maximum length = 64

auth_timeout<Integer>

The maximum number of seconds the system will wait for a response from the TACACS server.
Minimum value = 1
Maximum value =

accounting<Boolean>

Enable accounting in the tacacs server.

Read only properties

address_type<Integer>

Configuration Type. Values: 0: IPv4, 1: IPv6.
