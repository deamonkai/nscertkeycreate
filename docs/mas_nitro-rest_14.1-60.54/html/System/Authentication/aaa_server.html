aaa_server

Configuration for AAA Server configuration resource.

Read/write properties

external_servers<external_server[]>

List of cascaded external servers for single factor authentication. For two-factor authentication, list of primary external servers.

log_ext_group_info<Boolean>

Log external group info.

primary_server_name<String>

Name of primary server name.
Minimum length = 1
Maximum length = 128

two_factor_label<String>

Two-factor authentication label.
Maximum length = 128

fallback_local_authentication<Boolean>

Enable local fallback authentication.

primary_server_type<String>

Type of primary server. Supported types 1. LOCAL 2.RADIUS 3.LDAP 4.TACACS 5.KEYSTONE.
Minimum length = 1
Maximum length = 32

id<String>

Id is system generated key for all the servers.

secondary_external_servers<external_server[]>

List of secondary external servers for two-factor authentication.
