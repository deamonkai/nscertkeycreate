device_techsupport

Configuration for Techsupport for device resource.

Read/write properties

last_updated_time<Double>

Last Updated Time.

id<String>

Id is system generated key for techSupport bundle files.

ip_address<String>

Device IP Address.
Minimum length = 1
Maximum length = 64

techsupport_filesize<Double>

Techsupport file size.

is_upload_file<Boolean>

To upload the collector archive.

techsupport_password<String>

Techsupport Password.

encrypted<Boolean>

Indicates if the techsupport file is encrypted..

citrix_password<String>

Citrix password.

ns_auth_token<String>

Authorisation token required for uploading the file from NS.

ts_scope<String>

Scope can be cluster or node or partition.

ts_description<String>

Description for the file.

ts_time<String>

Time from which techSupport file will be generated.

citrix_username<String>

Citrix username.

ts_partition_name<String>

Name of the partition.

file_name<String>

file name.

case_or_service_request_number<String>

case or serrvice request number.

proxy_server<String>

Proxy server ip.

Read only properties

device_name<String>

Device Hostname.

techsupport_filename<String>

Name of the techsupport file.
