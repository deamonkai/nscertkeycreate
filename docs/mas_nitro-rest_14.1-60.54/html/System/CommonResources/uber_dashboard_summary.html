uber_dashboard_summary

Configuration for Uber Dashboard Summary resource.

Read/write properties

content<property_map[]>

List of content in key/value.

is_guided<Boolean>

Guided.

id<String>

Id is system generated key for all the tasks.

document_link<String>

Document Link.

last_modified_time<Double>

Last modified time in seconds.

weightage<Integer>

Weight of the task.
Maximum value =

categories<String>

Task Category.

is_acknowledged<Boolean>

Acknowledged.

is_completed<Boolean>

completed.

activity_id<String>

Activity ID is used when polled to get the update.
Maximum length = 128

poll_now<Boolean>

Option to trigger poll now.

categories_array<String[]>

Task Category in array format.

Read only properties

title<String>

Task Name.

task_name<String>

Task Name.

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
