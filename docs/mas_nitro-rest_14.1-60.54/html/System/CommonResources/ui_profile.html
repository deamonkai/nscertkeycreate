ui_profile

Configuration for UI Load profile resource.

Read/write properties

motd<String>

Message of the Day (contents of motd file) that can be shown on UI after successful login.

upgrade_agent_version<String>

Indicates the next version the agent needs to upgrade to..

deployment_type<String>

Indicates the type of deployment of NetScaler Console: standalone or ha or scaleout..

config_motd<Boolean>

Configure Message of the Day (contents of motd file), this needs to be set true if user wants to configure message if the day.

is_cloud<Boolean>

True if its a cloud deployment.

is_container<Boolean>

True if its a container deployment.

is_cuxip_enabled<Boolean>

Indicates whether CUXIP is enabled or not..

maps_apikey<String>

Maps API Key.

hist_mig_inprog<Boolean>

Indicates whether the historical tables database migration is in progress or not..

is_passive<Boolean>

Indicates the node's state: ACTIVE or PASSIVE in a HA deployment..

pending_tasks_count<Integer>

Count of incomplete tasks.

cc_profile_skip<Boolean>

True, if cloud identity authenticate is to be skipped.

agent_count<Integer>

Count of agents deployed.

cc_profile_id<String>

Returns cloud profile id..

cc_client_id<String>

Returns cloud profile client id..

cloud_deployment_env<String>

Cloud deployment environment..

cloud_platform<String>

Cloud platform..

device_count<Integer>

Count of devices.

is_light_tenant<Double>

Indicates whether the tenant is light or not.

cc_profile<Boolean>

True, if cloud profile is updated by user..

Read only properties

hostname<String>

Host name on which system is running.

current_time_formatted<String>

Current Time (Formatted).

time_zone<String>

Server Time Zone.

hostid<String>

Host Id.

host<String>

Host IP Address on which system is running, this will set for each client session only.

uptime<String>

Uptime.

hypervisor_uptime<String>

Hypervisor Uptime.

product<String>

Product Name.

platform<String>

Platform.

current_user_permission<String>

This property will show the permission type for current user.

time_zone_offset<Integer>

Time zone offset in minutes (Example:-330).

sysid<String>

System Id.

build_number_short<String>

Build Number without Date.

build_number<String>

Build Number.

serial<String>

Serial Number.

product_build_number<String>

Product Build Number.

current_tenant<String>

Current Tenant Name for logged-in user.

product_version<String>

Product Version.

current_tenant_id<String>

Current Tenant Id for logged-in user.

bios_version<String>

BIOS Version.

current_time<Integer>

Current Time.

username<String>

User Name who is currently connected to the system.
