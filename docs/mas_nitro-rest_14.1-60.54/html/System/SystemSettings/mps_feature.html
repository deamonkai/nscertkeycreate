mps_feature

Configuration for Feature Toggle resource.

Read/write properties

ops_toggle<Integer>

This is Ops controllable. 0: Disable UI and Backend, 1: Disable UI and enable Backend, 2: Enable UI and disable Backend, 3: Enable UI and Backend. Ops controlled takes higher precedence than Admin Controlled..
Maximum value =

admin_toggle<Integer>

This is Admin controllable. 0: Disable UI and Backend, 1: Disable UI and enable Backend, 2: Enable UI and disable Backend, 3: Enable UI and Backend..
Maximum value =

description<String>

Feature Description..

id<String>

Feature ID.

feature_name<String>

Feature Name.

built_in<Boolean>

This is Ops controllable and will not be visible to the Admin to control. If true: Ops controllable feature, false: Admin controllable feature..
