db_corruption_check

Configuration for DB Corruption Check resource.

Read/write properties

completion_percentage<Integer>

Percentage of completion of the database corruption scan script execution.

job_trigger_time<Integer>

Time when database corruption scan is started.

id<String>

Id is system generated key for all entries..

recommendation<String>

Recommendation based on the database corruption scan script execution.
Maximum length = 1024

scan_scope<String>

Scope of database corruption scan - CONFIG_TABLES for configuration data only, ALL_TABLES for complete database scan.
Maximum length = 256

status<String>

Status of database corruption scan e.g. SUCCESS, FAILED, In Progress.
Maximum length = 4096

job_trigger_end_time<Integer>

Time when database corruption scan is ended.

result<String>

Result of the database corruption scan script execution.
Maximum length = 4096
