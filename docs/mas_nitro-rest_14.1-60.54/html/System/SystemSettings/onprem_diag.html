onprem_diag

Configuration for OnpremDiag resource.

Read/write properties

adm<String>

check if is ADM reachable.
Minimum length = 1
Maximum length = 255

download_connectivity_status<Integer>

Status of download URL check. Returns 0 for success and non-zero for failure.

download_connectivity_error_details<String>

Error encountered during download connectivity check.

id<String>

ID of Security Advisory scan.

end_time<Double>

End time of diagnostics test.

is_user_triggered<Boolean>

is triggered by user.

safehaven_connectivity_status<Integer>

Status of Safehaven connectivity. Returns 0 for success and non-zero for failure.

status<String>

Diagnostics status.
Minimum length = 1
Maximum length = 255

safehaven<String>

check if is Safehaven reachable.
Minimum length = 1
Maximum length = 255

download<String>

Check if Download server is reachable.
Minimum length = 1
Maximum length = 255

safehaven_connectivity_error_details<String>

Error encountered during safehaven connectivity check.

start_time<Double>

Start time of diagnostics test.

managed_device_status<String>

Devices list with IP Address and status.
