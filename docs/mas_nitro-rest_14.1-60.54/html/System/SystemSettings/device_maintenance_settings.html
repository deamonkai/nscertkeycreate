device_maintenance_settings

Configuration for Device maintenance settings resource.

Read/write properties

from_time<Integer>

Start time of the maintenance window in seconds since the Unix Epoch.

created_time<Integer>

Time when the maintenance window setting was created in epoch.

to_time<Integer>

End time of the maintenance window in seconds since the Unix Epoch.

managed_device_id<String>

Id is system generated key for all entries..

comments<String>

device specific comment for the device maintenance window setting.

id<String>

Id is system generated key for all entries..

created_by<String>

info of user who configured the maintenance window.

active_status<String>

whether the device is still within the maintenance window or if it has been completed, possible values are scheduled, in-progress, completed.

last_update_time<Integer>

Time when the maintenance window setting was last updated in epoch.

managed_device_id_list<String[]>

Id is system generated key for all entries..
