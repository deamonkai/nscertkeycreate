mps_ccp_client

Configuration for MPS CCP client information resource.

Read/write properties

last_up_time<Double>

Last time when the On-prem Console was seen up.

id<String>

Id is system generated key for CCP client registered with NMX Cloud..

version<String>

version..

instance_id<String>

Instance ID from CC Profile.

enable_telemetry<Boolean>

enable telemetry.

name<String>

Agent IP Address.

customer_name<String>

customer display name.

state<String>

Agent state.
