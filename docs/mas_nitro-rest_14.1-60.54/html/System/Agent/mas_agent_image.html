mas_agent_image

Configuration for NetScaler Agent Image resource.

Read/write properties

file_location_path<String>

File Location on Client for upload/download.
Minimum length = 1

file_name<String>

File Name.
Minimum length = 1
Maximum length = 256

proxy_server_ip_address<String>

IP address of proxy server.
Minimum length = 1
Maximum length = 64

proxy_server_port<Integer>

Port number of proxy server.

application_id<String>

Agent application ID.

k8_helmchart<String>

k8 helm chart.
Minimum length = 1
Maximum length = 128

k8_agent_version<String>

Kubernetes Agent Version.

proxy_server_username<String>

Username for proxy server.
Minimum length = 1
Maximum length = 128

password<String>

Agent password.

proxy_server_password<String>

Password for proxy server.
Minimum length = 1
Maximum length = 128

k8_yaml_filename<String>

k8 yaml filename.
Minimum length = 1
Maximum length = 128

Read only properties

file_last_modified<String>

Last Modified.

file_last_modified_epoch<Double>

Last Modified (Epoch).

file_size<Integer>

file_size.
