aa_web_anomaly_limits_l2

Configuration for Temporary table for AA Web Anomaly IQR limits resource.

Read/write properties

apt_lower_lim<Double>

App Processing Time Lower Limit.

spt_lower_lim<Double>

Server Processing Time Lower Limit.

iqr_ctnsappname<String>

AppName.
Maximum length = 255

iqr_application_name<String>

Application Name.
Maximum length = 255

snl_lower_lim<Double>

Server Network Latency Time Lower Limit.

iqr_ip_address<String>

IP Address.
Maximum length = 255

cnl_lower_lim<Double>

Client Network Latency Time Lower Limit.

cnl_upper_lim<Double>

Client Network Latency Time Upper Limit.

snl_upper_lim<Double>

Server Network Latency Time Upper Limit.

spt_upper_lim<Double>

Server Processing Time Upper Limit.

apt_upper_lim<Double>

App Processing Time Upper Limit.
