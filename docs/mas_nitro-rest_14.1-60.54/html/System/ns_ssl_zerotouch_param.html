ns_ssl_zerotouch_param

Configuration for Used to configure SSL zero touch param on NetScaler resource.

Read/write properties

public_key<String>

Public key generated on the device where SSL zero touch param is configured.

managed_device_id<String>

Id of the device where SSL zero touch param is enabled.

passphrase<String>

Passphrase used to configure SSL zero touch param on NetScaler.
Minimum length = 1
Maximum length = 127

ip_address<String>

IP Address of the device where SSL zero touch param is enabled.
Minimum length = 1
Maximum length = 64

id<String>

Id is system generated key.

isCACertificatePresent<Boolean>

Set to true, if the Console CA certificate is already present.

ns_ip_address_arr<String[]>

List of NetScaler IP Address where SSL zero touch param is to be configured.

file_name<String>

CA file name..
Minimum length = 1
Maximum length = 128

ns_id_arr<String[]>

List of NetScaler IDs where SSL zero touch param is to be configured.

eligible_devices<String[]>

List of NetScaler IDs eligible for SSL zero touch param configuration.

file_data<String>

Base64 encoded CA cert file content.
Maximum length = 50000

Read only properties

activity_id<String>

Activity ID that is used by GUI to track the status of configuring SSL zero touch param on NetScaler..
