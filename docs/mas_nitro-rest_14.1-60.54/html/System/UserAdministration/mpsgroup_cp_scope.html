mpsgroup_cp_scope

Configuration for Configpack Scope for MPSGroup resource.

Read/write properties

task_type<String>

Type of the task: Stylebook or Configpack.

group_id<String>

Id of the group.

sb_id<String>

Stylebook ID.

id<String>

Auto generated ID..

configpack_id<String>

ID of the configpack.

operation_status<String>

Status of the operation performed on the configpack.

status<String>

Status of the configpack.

operation<String>

Operation performed on the configpack.

sbuser<String>

User who performed the operation.

last_modified_datetime<String>

Last modified datetime.

sb_info<String>

Stylebook details.
