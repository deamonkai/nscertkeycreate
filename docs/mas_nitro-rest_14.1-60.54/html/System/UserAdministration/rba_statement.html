rba_statement

Configuration for RBA Statement defines the permission for a resource resource.

Read/write properties

operation_name<String>

Operation for a resource.

access_type<Boolean>

true, if access is allowed.

id<String>

Id is system generated key for all the permission statements.

dependent_resources<String>

Comma separated names of resources, permission of these resources is dependent upon permission of whatever is given in resource_type..

Resource Type.
