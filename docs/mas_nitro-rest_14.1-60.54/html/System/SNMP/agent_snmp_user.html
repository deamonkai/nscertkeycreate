agent_snmp_user

Configuration for Agent SNMP User resource.

Read/write properties

name<String>

Name of SNMP User.
Minimum length = 1
Maximum length = 32

privacy_protocol<Integer>

Privacy Protocol of SNMP User. Values: 0:noValue, 1: DES, 2: AES.
Maximum value =

auth_password<String>

Authentication Password of SNMP User.
Minimum length = 8
Maximum length = 32

auth_protocol<Integer>

Authentication Protocol of SNMP User. Values: 0:noValue, 1: MD5, 2: SHA1.
Maximum value =

agent_id<String>

The ID of Agent.
Maximum length = 128

id<String>

Id is system generated key.

security_level<Integer>

Security Level of SNMP User. Values: 0: noAuthNoPriv, 1: authNoPriv, 2: authPriv.
Maximum value =

view_name<String>

SNMP View Name attached to the SNMP User.
Maximum length = 32

privacy_password<String>

Privacy Password of SNMP User.
Minimum length = 8
Maximum length = 32
