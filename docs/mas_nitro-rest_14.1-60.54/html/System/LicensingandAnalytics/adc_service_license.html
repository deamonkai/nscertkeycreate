adc_service_license

Configuration for NetScaler Service License Entitlement Information resource.

Read/write properties

serial_number<String>

SerialNumbers.

start_date<String>

Start date of license subscription.

approved<Boolean>

Approved.

id<String>

Id is system generated key.

entitlement_id<String>

Entitlement Id for the license file.

entitlement_action<String>

Action of the entitlement requested (Extend/Deliver/Update/Cancel).

order_item_id<String>

Order line number, this is required for multiple orders.

properties<String>

Misc content goes here.

end_date<String>

Expiry date of license subscription.

customer_id<String>

Customer Id for the tenant who has purchased the license.

quantity<String>

Number of entitlements of the type purchased.

edition<String>

Edition of the entitlement.

order_id<String>

Id is system generated key.

product_sku<String>

Product SKU of the License which specifies the type of license.

sku_intent<String>

Intent of the product SKU (test/final).

entitlement_type<String>

Entitlement Type eg. ProductionTrial.

lic_state<Integer>

0 = Trial Period, 1 = Licensed, 2 = Grace period, 3 = Unlicensed, 4 = Canceled.

Read only properties

entitled_dns_queries<String>

Max entitled dns queries.

entitled_data<String>

Maximum data usage Entitled by license.

entitled_storage<String>

Maximum Storage for the license in GB.

total_quantity<Integer>

Max entitled quantity.

total_entitled_storage<Integer>

Max entitled storage.

total_entitled_dns_queries<Integer>

Max Entitled DNS queries.

total_entitled_data<Integer>

Maximum data usage Entitled by license.
