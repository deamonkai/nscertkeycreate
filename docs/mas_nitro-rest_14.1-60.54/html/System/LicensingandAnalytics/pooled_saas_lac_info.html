pooled_saas_lac_info

Configuration for NetScaler Pooled Saas license access codes info resource.

Read/write properties

customer_id<String>

Customer Id for the tenant who has purchased the license.

activation_id_list<String>

All activation IDs of the LAC.

license_access_codes<String>

ADS pooled license access codes.

id<String>

Id is system generated key.

lac_last_proc<String>

Last scan of lac in UTC.

properties<String>

All properties.

lac_proc_state<String>

LAC processing state.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
