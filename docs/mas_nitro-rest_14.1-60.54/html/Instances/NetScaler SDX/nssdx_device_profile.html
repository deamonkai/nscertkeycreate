nssdx_device_profile

Configuration for NetScaler SDX Device Profile resource.

Read/write properties

use_global_setting_for_communication_with_ns<Boolean>

True, if the communication with Instance needs to be global and not device specific.

id<String>

Id is system generated key for all the device profiles.

svm_ns_comm<String>

Communication protocol (http or https) with Instances.
Minimum length = 1
Maximum length = 10

type<String>

Profile Type, This must be with in specified supported instance types: blx,ns,nssdx,cpx.
Minimum length = 1
Maximum length = 128

name<String>

Profile Name.
Minimum length = 1
Maximum length = 128

snmpversion<String>

SNMP version for this profile.

snmpauthpassword<String>

SNMP v3 auth password for this profile.
Minimum length = 8
Maximum length = 31

snmpcommunity<String>

SNMP community for this profile.
Maximum length = 31

https_port<Integer>

HTTPS port to connect to the device.

snmpprivpassword<String>

SNMP v3 priv password for this profile.
Minimum length = 8
Maximum length = 31

snmpsecuritylevel<String>

SNMP v3 security level for this profile.

host_password<String>

Host Password for this profile.Used for BLX form factor of ADC.
Minimum length = 1
Maximum length = 127

passphrase<String>

Passphrase with which private key is encrypted.

sync_operation<Boolean>

Flag to indicate whether operation should be performed in sync/async mode.

snmpsecurityname<String>

SNMP v3 security name for this profile.
Maximum length = 31

ssh_port<String>

SSH port to connect to the device.

host_username<String>

Host User Name for this profile.Used for BLX form factor of ADC.
Minimum length = 1
Maximum length = 127

ssl_private_key<String>

SSL Private Key for key based authentication.

snmpprivprotocol<String>

SNMP v3 priv protocol for this profile.

max_wait_time_reboot<String>

Max waiting time to reboot NetScaler.

snmpauthprotocol<String>

SNMP v3 auth protocol for this profile.

username<String>

Instance credentials.Username provided in the profile will be used to contact the instance.
Minimum length = 1
Maximum length = 127

password<String>

Instance credentials.Password for this profile.
Minimum length = 1
Maximum length = 127

http_port<Integer>

HTTP port to connect to the device.

cb_profile_name<String>

Profile Name, This is one of the already created NetScaler profiles.

ssl_cert<String>

SSL Certificate for certificate based authentication.

ns_profile_name<String>

Profile Name, This is one of the already created NetScaler profiles.

Read only properties

tenant_id<String>

Tenant Id of device profile.

is_default<Boolean>

True, if this profile is system generated and can not be deleted.

is_backup<Boolean>

true, if this profile is a generated back up profile of user created profile.

act_id<String>

Activity Id.
