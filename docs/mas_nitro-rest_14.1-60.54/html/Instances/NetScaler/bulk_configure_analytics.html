bulk_configure_analytics

Configuration for Bulk configuration of analytics resource.

Read/write properties

bot_insights<Boolean>

Configure Bot Insights for all vips.

ns_ip_address_regex<String>

ns_ip_address_Regex.

auto_enable_analytics<Boolean>

Save and apply this config for all the future discovered applicable vservers.

client_side_measurement<Boolean>

Configure Client Side Measurement for all vips.

waf_insights<Boolean>

Configure WAF Insights for all vips.

policy_name<String>

Policy Name used for naming a Global Analytics config.
Maximum length = 128

security_insights<Boolean>

Configure Security Insights for all vips.

gateway_insights<Boolean>

Configure Gateway Insights for all vips.

type<String>

.

hdx_insights<Boolean>

Configure HDX Insights for all vips.

web_insights<Boolean>

Configure WebInsights for all vips.

id<String>

Primary key for this table..

ctnsappname_regex<String>

ctnsappname_Regex.

tenant_id<String>

Tenant ID.

ns_ip_address<String>

NS IP Address.

ns_ip_address_arr<String[]>

List of NetScaler IP Address.
Minimum length = 1
Maximum length = 256

gw_enablement<Boolean>

Flag to indicate if request is for GW Analytics.

analytics_op<String>

Analytics operation enable/disable.

Read only properties

Activity Id.
