autoscale_activity_status

Configuration for Activity Status for the Autoscale operations resource.

Read/write properties

availability_zone<String>

Availability Zone.
Minimum length = 1
Maximum length = 128

asg_id<String>

AutoSclae group ID to which the cluster is associated.

id<String>

Id is system generated key for all the task logs.

task_id<String>

task_id returned from other component like Provisioning/Stylebook/DistMgr.

starttime<Double>

Start Time in microseconds.

activity_id<String>

Parent Activity Id: Same for all the parallel provisioning processes for a particular autoscale group.

az_activity_id<String>

Availability Zone Activity Id.

status<String>

Status of the operation.

errorcode<Integer>

Error code in case of failure.

Read only properties

severity<String>

Error severity.

activity<String>

Activity details.

message<String>

Message provided during the operation.

is_last<Boolean>

Enabled for the last operation.
