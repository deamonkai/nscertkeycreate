ns_ns_ip

Configuration for IP on NetScaler resource.

Read/write properties

netmask<String>

netmask.
Minimum length = 1
Maximum length = 64

ns_ip_address<String>

NetScaler IP Address where IP needs to be created.
Minimum length = 1
Maximum length = 64

state<String>

state ENABLED/DISABLED.
Minimum length = 1
Maximum length = 64

id<String>

Id is system generated key.

mgmtaccess<String>

mgmtaccess ENABLED/DISABLED.
Minimum length = 1
Maximum length = 64

ipaddress<String>

NetScaler ipaddress that needs to be created.
Minimum length = 1
Maximum length = 64

hostname<String>

Hostname of the managed device.
Minimum length = 1
Maximum length = 256

type<String>

Type of IP Address, Possible Values: SNIP, NSIP, VIP.
Minimum length = 1
Maximum length = 64

save_config<Boolean>

true, if save config is required.

Read only properties

partition_name<String>

Partition Name.
