adc_vserver_appflow_info

Configuration for Virtual server appflow info on NetScaler resource.

Read/write properties

name<String>

Virtual server name.
Maximum length = 255

transport_mode<String>

Transport Mode [ Logstream/Appflow ].

export_option<String[]>

Export Options: TCP, ICA.

httpxforwardedfor<String>

X-Forwarded-For Header Export status.

ns_ip_address<String>

NetScaler IP Address.
Minimum length = 1
Maximum length = 64

transaction_log_effective<String>

Transaction Log Enabled/Disabled.

client_header<String>

Custom Client Header to Export.

appflow_policy_rule<String>

Appflow policy rule.

type<String>

Tells whether virtual server type.
Minimum length = 2
Maximum length = 64

netprofile<String>

Net Proile.

es4nslog<String>

ESNS enable.

Read only properties

partition_name<String>

Partition Name.
