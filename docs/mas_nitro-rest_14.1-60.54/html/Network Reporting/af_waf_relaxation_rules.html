af_waf_relaxation_rules

Configuration for WAF relaxation rules resource.

Read/write properties

max_value<Integer>

Maximum length of the filed to be relaxed.

http_req_url<String>

HTTP Request URL to be relaxed.
Maximum length = 2000

ctnsappname<String>

ctnsappname.
Maximum length = 255

is_system<Boolean>

Set to true when system generated and false for user configured rule.

slack_profile<String>

Slack Profile name.

field_type<String>

Field type to be relaxed.
Maximum length = 512

profile_name<String>

profile_name.
Maximum length = 32

learning_mode<Integer>

Learning mode to differentate between a rule to be pruned (2) or deployed (0 or 1).

status<Boolean>

Status is true when the rule is enabled and false when the rule is disabled.

mail_profile<String>

Mail Profile name.

user_name<String>

Field used to maintain the name of User who take an action that lead to this transaction record.

sms_profile<String>

SMS Profile name.

is_auto_deploy<Boolean>

Field to check if auto deploying is ENABLED/DISABLED.

is_deployed<Integer>

Field refelecting status of rule i.e 0-ready to deploy,1-deployed,2-skipped,4-failed.

regex<Integer>

Regex number 1 is REGEX and 2 is NOTREGEX in bind appfw profile per security check.

resourceid<String>

Resource Id generated for each relaxation rule which is bound to appfw profile..
Minimum length = 1
Maximum length = 128

mode<Integer>

Deployment mode would be OPEN(0) / CLOSE(1).

min_value<Integer>

minimum length of the field to be relaxed.

value_type<String>

Value type to be relaxed.
Maximum length = 512

value_expr<String>

Value expression to be relaxed.
Maximum length = 512

violation_location<Integer>

Violation Location.

count_value<Integer>

count for the number of times this rule has caused violation.

servicenow_profile<String>

servicenow Profile name.

si_device_ip_address<String>

NetScaler IP Address..
Minimum length = 1
Maximum length = 512

adm_learn_profile_name<String>

adm_learn_profile_name.
Minimum length = 1
Maximum length = 32

grace_period<Double>

time before a rule is auto deployed.

violation_id<Integer>

Violation Category.

rpt_sample_time<Double>

Report Sample time..

field_name<String>

Field name to be relaxed.
Maximum length = 512

no_of_violations<Double>

Number of violations.

grace_period_expiry<Double>

Number of seconds left to trigger autodeploy for relaxation of URL.

comment<String>

Comment to be added to a relaxation rule while deploying it.
Maximum length = 255

Read only properties

id<String>

Id is system generated key..
