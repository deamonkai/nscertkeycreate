perf_custom_dashboard

Configuration for Custom Dashboard for Network reports resource.

Read/write properties

entity_type<String>

Entity Type.

name<String>

name.
Minimum length = 1
Maximum length = 128

device_family<String>

Device Family of which this custom dashboard belongs to.

description<String>

Custom dashboard description.

reporting_category<String>

Reporting Category.

no_of_columns<Integer>

No of columns in dashboard.
Minimum value = 1
Maximum value =

id<String>

Id is system generated key for all the perf custom dashboard configuration .
Maximum length = 256

timestamp<Double>

timestamp in seconds.

agg_level<String>

Aggregation Level.

instances_array<String[]>

Instances in array.

coordinates_array<String[]>

Co-ordinates in array.

entities_array<String[]>

Entities (Virtual servers or Network Interfaces).

report_names_array<String[]>

Report names in array.

Read only properties

instances<String>

Display Names of NetScaler instances.

entities<String>

Entities(virtual servers or network interfaces) with instance display_names.

report_names<String>

Report Names.

coordinates<String>

Co-ordinates.
