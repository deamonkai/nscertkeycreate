config_command

Configuration for Configuration Commands for use in configuration template resource.

Read/write properties

nitro_payload<String>

NITRO Request Payload.

nitro_resource<String>

NITRO Resource Name.
Minimum length = 1
Maximum length = 128

index<Integer>

Index that stores the position of the command in a command array.

rollback_command<String>

Rollback Command String.

command<String>

Command String for Protocols SSH|SCP|SFTP.

id<String>

Id is system generated key for all config command.

nitro_method<String>

NITRO Request Method: POST|PUT|DELETE.
Minimum length = 1
Maximum length = 32

parent_name<String>

Parent name of config command.

protocol<String>

Protocol used by Configuration Command: SSH|SCP|SFTP.
Minimum length = 1
Maximum length = 32

parent_id<String>

Parent Id of config command.

timeout<Integer>

Command Timeout in secs.
Maximum value =
