external_server

Configuration for External Server configuration resource.

Read/write properties

external_server_name<String>

Name of external server.
Minimum length = 1
Maximum length = 128

priority<Integer>

Priority of external server.
Minimum value = 2
Maximum value =

id<String>

Id is system generated key for all the external servers.

is_secondary_external_server<Boolean>

If true, external server is configured for secondary server authentication.

external_server_type<String>

Type of external server. Supported types 1.RADIUS 2.LDAP 3.TACACS 4.KEYSTONE.
Minimum length = 1
Maximum length = 32
