syslog_server

Configuration for Syslog Server resource.

Read/write properties

ip_address<String>

Syslog server IP address.
Minimum length = 1
Maximum length = 64

log_level_none<Boolean>

Send no logs to this syslog server.

log_level_critical<Boolean>

Send logs of level critical to this syslog server.

log_level_warning<Boolean>

Send logs of level warning to this syslog server.

shell_log_levels<String[]>

Events with these log levels pertaining to the shell facilities(local6 & local7) will be sent to this syslog server.

enable_shell_logging<Boolean>

Toggle to specify whether event logs pertaining to the shell facilities(local6 & local7) shall be sent to this syslog server.

log_level_all<Boolean>

Send logs of all levels to this syslog server.

log_level_error<Boolean>

Send logs of level error to this syslog server.

port<Integer>

Syslog server port.
Maximum value =

log_level_info<Boolean>

Send logs of level info to this syslog server.

name<String>

Syslog server name.
Minimum length = 1
Maximum length = 128

Read only properties

log_levels<String>

Set of all log levels at which messages are sent to this syslog server.

address_type<Integer>

Configuration Type. Values: 0: IPv4, 1: IPv6.
