af_alert_l3

Configuration for Threshold breach information for Level 3 resource.

Read/write properties

threshold_value<String>

Threshold Value.

is_enable_flag<Double>

Only applicable for configuration entries, 1=enable, 2=disable, 3=delete.

threshold_name<String>

Threshold Name.

violation_count<Double>

No of times violation happens for particular entity.

reference_key<String>

Reference Key.

entity<String>

Entity.

city<String>

City.
Maximum length = 255

ctnsappname<String>

AppName.
Maximum length = 255

duration<String>

Threshold calculation frequency period daily,weekly etc.

metric<String>

Metric.

rpt_sample_time<Double>

Report Sample time..

tenant_name<String>

Tenant Name.

region<String>

Region Code.
Maximum length = 10

ip_address<String>

Field to store the ip address as it is along with any extension.
Maximum length = 512

breach_value<String>

Breach Value.

exporter_id<Double>

Exporter Id.

country<String>

Country Code.
Maximum length = 8

entity_type<String>

Entity Type.

stylebook_flag<Boolean>

Stylebook threshold configuration true or false.
