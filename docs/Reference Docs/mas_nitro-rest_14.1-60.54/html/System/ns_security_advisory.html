ns_security_advisory

Configuration for Security advisory details on NetScaler Console resource.

Read/write properties

overall_scan_status<String>

Overall status of the scan.
Minimum length = 1
Maximum length = 128

ns_ip_addresses_list<String>

List of NetScaler IP Address.

scan_type<String>

Type of scan.
Minimum length = 1
Maximum length = 128

end_time<Double>

Scan end time.

scan_recurrence<String>

Scan recurrence.
Minimum length = 1
Maximum length = 128

scan_execution_time<Double>

Scan execution time given by the user.

scan_initiated_by<String>

User who initiated the scan.
Minimum length = 1
Maximum length = 128

secure_config_advisory_status<String>

Secure Config Advisory Status.
Minimum length = 1
Maximum length = 256

cve_detection_status<String>

CVE Detection Status.
Minimum length = 1
Maximum length = 256

start_time<Double>

Scan start time.

scan_all_devices<Boolean>

Boolean to indicate if scan has to be run on all devices.

file_integrity_monitoring_status<String>

File Integrity Monitoring Status.
Minimum length = 1
Maximum length = 256

id<String>

ID of Security Advisory scan.

next_scan_time<Double>

Next System Scan Time.

file_integrity_monitoring<Boolean>

File Integrity Monitoring.

enable_cve_detection_system_scan<Boolean>

Enable CVE Detection System Scan.

cve_detection<Boolean>

CVE Detection.

enable_secure_config_advisory_system_scan<Boolean>

Enable Secure Config Advisory System Scan.

secure_config_advisory<Boolean>

Secure config advisory.

ns_ip_addresses<String[]>

NetScaler IP Addresses.

enable_file_integrity_monitoring_system_scan<Boolean>

Enable File Integrity Monitoring System Scan.
