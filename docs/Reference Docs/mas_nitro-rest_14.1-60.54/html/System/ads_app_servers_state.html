ads_app_servers_state

Configuration for ADS Application Server State resource.

Read/write properties

server_state<String>

State of server.

monitortotalprobes<Integer>

Total number of probes by Monitor.

availability_zone<String>

Availability Zone.

customer_name<String>

Customer name.

lastresponse<String>

Last Response from the server.

monitortotalfailedprobes<Integer>

Total number of failed probes by Monitor.

monitorcurrentfailedprobes<Integer>

Current number of failed probes by Monitor.

monitor_state<String>

Current state of the Monitor.

svc_grp_name<String>

Service Group Name of the Server.

poll_time<Integer>

Last Polling Time.

ns_ip_address<String>

NetScaler Console Name.

port<Integer>

Server Port.

monitor_name<String>

Monitor Name.

server_name<String>

IP Address of the Server.
