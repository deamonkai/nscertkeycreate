mps_agent_diag

Configuration for MPS Agent diag information resource.

Read/write properties

geo<String>

agent pop geo information.

download<Integer>

Agent diag download.citrixnetworkapi.net state.

pop<String>

agent pop information.

trust<Integer>

Agent diag trust.citrixnetworkapi.net state.

proxy<Integer>

Agent diag proxy configured or not.

cas<Integer>

Agent diag cas endpoint state.

dns<Integer>

Agent diag DNS state.

s3<Integer>

Agent diag s3 endpoint state.

endtime<String>

endtime of the agent diag.

id<String>

Id is the agent id.

grp<Integer>

Agent diag adm.cloud.com state.

adm<Integer>

Agent diag adm.cloud.com state.

agent<Integer>

Agent diag adm.cloud.com state.
