remote_backup_config

Configuration for remote backup configuration resource.

Read/write properties

db_health<String>

Database health of the node.

id<String>

Id is system generated key for all the system users.

backup_vm_ip_address<String>

BACKUP VM IP Address.

sync_progress<String>

Database synchronization status with the primary node.

retention_interval<Integer>

No of days upto which the base backup will be retained.
Minimum value = 5
Maximum value =

is_enabled<Boolean>

Is Remote backup enabled for this server.

Read only properties

disk_usage<Double>

Disk Usage (%).

memory_usage<Double>

Memory Usage (%).

disk_total_capacity<Double>

Disk Total Capacity in KB.

memory_free<Double>

Free memory available in KB.

memory_total<Double>

Total memory in KB.

disk_total<Double>

Total available disk space in KB.

cpu_usage<Double>

CPU Usage (%).

disk_used<Double>

Disk used in KB.

disk_free<Double>

Free disk available in KB.

act_id<String>

Activity Id.
