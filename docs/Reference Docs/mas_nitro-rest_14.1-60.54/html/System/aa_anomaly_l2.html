aa_anomaly_l2

Configuration for AA Anomaly Report table for Level 2 resource.

Read/write properties

ip_address<String>

IP Address.
Maximum length = 512

ctnsappname<String>

AppName.
Maximum length = 255

metric_value<Double>

metric_value in sampled timeframe..

svcname<String>

Source IP Address.
Maximum length = 255

reason<String>

Reason..
Maximum length = 255

rpt_sample_time<Double>

Report Sample time..

metric_name<String>

metric_name in UnevenLB, serverDelay.
Maximum length = 255

usecase<String>

usecase name.
Maximum length = 255
