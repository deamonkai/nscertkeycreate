export_report

Configuration for Export report resource.

Read/write properties

recurrence<String>

Recurrence (Daily, Weekly, Monthly)..
Maximum length = 16

export_format<String>

Report Export format (PDF/JPEG/PNG).
Minimum length = 3
Maximum length = 4

report_title<String>

Report Title.
Maximum length = 512

export_time<String>

Report Export time epoch (string representation of 11 digit numbers)..
Minimum length = 10
Maximum length = 16

mail_profile_name<String>

Mail profile name to send notification mail..
Maximum length = 255

recurrence_option<String>

Recurrence option (Weekly -> mon - sun, Monthly -> date in DD format)..
Maximum length = 128

slack_profile_name<String>

Slack profile name to send slack notification..
Maximum length = 255

subject<String>

subject.
Maximum length = 255

records_limit<Integer>

Record limit for csv export.

description<String>

Report Description..
Maximum length = 512

tabular_report<Boolean>

Tabular report value true and snapshot report value false..

page_name<String>

Page name..
Minimum length = 1
Maximum length = 255

export_now<Boolean>

Export now.

file_name<String>

Report File name..
