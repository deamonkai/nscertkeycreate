adm_manage_unsubscribe_email

Configuration for Unsubscribe Email resource.

Read/write properties

id<String>

Id is system generated key for adm_manage_unsubscribe_email .
Maximum length = 256

unsubscribe_email_notification<Boolean>

Set the value to true if the customer doesn't want to receive email notification..

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
