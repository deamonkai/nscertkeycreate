agent_snmp_mib

Configuration for Agent SNMP MIB Information resource.

Read/write properties

custom_id<String>

Custom identification number for appliance.
Minimum length = 1
Maximum length = 127

id<String>

Id is system generated key.

agent_id<String>

The ID of Agent.
Maximum length = 128

location<String>

Physical location of appliance.
Minimum length = 1
Maximum length = 127

contact<String>

Name of the administrator for appliance..
Minimum length = 1
Maximum length = 127

name<String>

Name for appliance.
Minimum length = 1
Maximum length = 127
