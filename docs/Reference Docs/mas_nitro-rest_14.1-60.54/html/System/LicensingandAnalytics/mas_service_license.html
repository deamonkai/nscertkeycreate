mas_service_license

Configuration for NetScaler Console License Information resource.

Read/write properties

start_date<String>

Start date of license subscription.

approved<Boolean>

Approved.

end_time<Double>

License expires at this timestamp.

id<String>

Id is system generated key.

entitlement_id<String>

Entitlement Id for the license file.

end_date<String>

Expiry date of license subscription.

customer_id<String>

Customer Id for the tenant who has purchased the license.

start_time<Double>

License validity starts from this timestamp.

serial_numbers<String>

SerialNumbers.

quantity<String>

Number of Licenses of the type purchased.

lic_type<Integer>

0 = Trial Period, 1 = Licensed, 2 = Grace period, 3 = Unlicensed, 4 = Canceled.

entitlement_type<String>

Entitlement Type eg. ProductionTrial.

product_sku<String>

Product SKU of the License which specifies the type of license : VIPs, Storage or ThirdParty VIPS.

Read only properties

max_tp_vservers<String>

Maximum Third Party Vservers.

max_vips<String>

Maximum VIPs.

max_storage<String>

Maximum Storage for the license in GB.

total_allowed_vips<Integer>

Total Allowed VIPs.

total_allowed_tp_vips<Integer>

Total Allowed Third Party Vservres.

days_to_expire<Integer>

Days to expire.

entitled_db_storage<Double>

Total entitled DB storage.
