database_storage_management

Configuration for Resource to fetch database storage management resource.

Read/write properties

truncate_in_progress<Boolean>

Indicate whether an user initiated truncate operation is already in progress or not.

sources_for_truncate<String[]>

Array of sources whose tables need to be truncated to free up database space..

allowed_truncate_count<Integer>

Remaining number of times data can be truncated, to restart ingestion in a calendar month.

truncate_id<String>

Unique ID which represents a specific truncate operation initiated by the user.

is_truncate_allowed<Boolean>

Indicate whether persisted historical data from various sources can be truncated or not..

maximum_truncate_count<Integer>

Maximum number of times data can be truncated, to restart ingestion in a calendar month.

data_ingestion<String>

Indicates whether data ingestion is enabled or not. Possible values: Enabled, Disabled.
