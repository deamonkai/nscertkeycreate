device_backup_policy

Configuration for Device Backup Policy resource.

Read/write properties

backup_ontrap_delay<Integer>

Delay in taking Backup on Receiving NetscalerConfigSave trap.

backup_on_cfg_sav_trap<Boolean>

Backup on receiving NetscalerConfigSave trap.

number_of_backups<Integer>

Number of backup files maintained per device.
Minimum value = 1

enable_external_transfer<Boolean>

Enable transfer of device backup files to external server.

encrypt_backup_file<Boolean>

Encrypts backup files.

enable_scheduled_backups<Boolean>

Enable Scheduled Backup (default is enabled).

is_interval_based<Boolean>

Parameter to decide whether backup setting is interval based or time based.

backup_times<String>

Comma Separated string of backup times to be scheduled..

sdx_backup_timeout<Integer>

Backup Timeout for SDX Backup.

polling_interval<Integer>

Frequency of device backup in hours.

enable_geodb_backups<Boolean>

Enable GeoDB Backup from NetScaler (default is disabled).

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
