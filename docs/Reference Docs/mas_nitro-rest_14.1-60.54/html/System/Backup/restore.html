restore

Configuration for Restore resource.

Read/write properties

required_ns<Integer>

Total number of instances present in backup file..

backup_file_name<String>

Backup file name.
Maximum length = 64

restore_type<Integer>

.

ip_address_list<String[]>

List of VM IP Address.
Minimum length = 1
Maximum length = 1024

sbi_file<String>

SDX Image selected by User for Restore.

required_throughput<Double>

Throughput required by the backedup instances to get restored..

max_allowed_ns<Integer>

Number of NetScaler instances current license allows to be restored..

backup_components<backup_component[]>

Backup Components.

max_allowed_throughput<Double>

Maximum throughput current license allows..

is_valid_license<Boolean>

To indicate if available license is valid and (NS+throughput) sufficient for restore..

backup_password<String>

Password of encrypted backup file.

Read only properties

premium_throughput<Double>

Throughput from backup under premium license..

max_allowed_standard_throughput<Double>

Maximum allowed throughput under existing standard license..

advanced_throughput<Double>

Throughput from backup under advanced license..

is_pooled_license<Boolean>

Flag to indicate that current SDX is operating in pooled license.

act_id<String>

Activity Id.

max_allowed_advanced_throughput<Double>

Maximum allowed throughput under existing advanced license..

standard_throughput<Double>

Throughput from backup under standard license..

max_allowed_premium_throughput<Double>

Maximum allowed throughput under existing premium license..

Use this query-parameter to get the restore resources in chunks.
