system_settings

Configuration for System Settings resource.

Read/write properties

secure_access_only<Boolean>

Secure Access only.

session_timeout_unit<String>

Session timeout unit for the system.

enable_certificate_download<Boolean>

Enable Certificate Download.

authorize_deviceapiproxy<Boolean>

Authorize the DeviceAPIProxy request.

is_metering_enabled<Boolean>

Enable Metering for NetScaler VPX on SDX.

session_timeout<Integer>

Session timeout for the system.

keep_alive_ping_interval<Integer>

Agent web socket keep alive ping interval for the system.

disable_agent_old_password_input<Boolean>

Disable old password input requirement while changing NetScaler Agent password.

enable_shell_access<Boolean>

Enable Shell access for non-nsroot User(s).

prompt_creds_for_stylebooks<Boolean>

Prompt Credentials for Stylebooks.

enable_delete_interface_on_adc<Boolean>

Flag to enable/disable deleting interface from ADCs on SDX.

enable_session_timeout<Boolean>

Enables session timeout feature.

heartbeatinterval<Integer>

Client heart beat interval in seconds.
Minimum value = 30
Maximum value =

svm_ns_comm<String>

Communication with Instances.
Minimum length = 1
Maximum length = 10

inventoryrefreshinterval<Integer>

Client inventory refresh interval in minutes.
Minimum value = 45
Maximum value =

enable_apiproxy_credentials<Boolean>

Enable API Proxy Credentials.

enable_cuxip<Boolean>

Used to enable/disable CUXIP(Customer User Experience Improvement Program).

enable_nsrecover_login<Boolean>

This setting enalbes nsrecover login for SVM.

id<String>

Id is system generated key.

basicauth<Boolean>

Allow Basic Authentication Protocol.

keep_adc_image_count<Integer>

Count for number of NetScaler images to be saved in Agent.

disk_utilization_threshold<Integer>

Disk utilization threshold after which data processing it stopped.

Read only properties

port_model<Integer>

Port Model for BR instance.

act_id<String>

Activity Id.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
