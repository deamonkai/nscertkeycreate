cc_profile

Configuration for Cloud Profile resource.

Read/write properties

tenant_id<String>

Tenant Id of cloud profile.
Minimum length = 1
Maximum length = 128

id<String>

Id is system generated key for all the cloud profiles.

cc_id<String>

Cloud Connect ID stored at server side.

enable_service_now<Boolean>

Enable Service Now feature.

Read only properties

enable_mastlmy<Boolean>

Enable NetScaler Console telemetry feature.

is_set<Boolean>

True, if this profile is updated by user.

svc_name<String>

Cloud service Name.

new_with_cloudconnect<Boolean>

True, if this profile is updated by user with cloudconnect.

skip_setting<Boolean>

True, if this setting is skipped by user.

customer_id<String>

Customer ID of the cloud profile holder.

instance_id<String>

Instance ID of the system.

customer_name<String>

Customer Name.

svc_end_point<String>

Service end point of the service.

pop_name<String>

Cloud service POP Name.

verify_url<String>

Verify URL.

act_id<String>

Activity Id.
