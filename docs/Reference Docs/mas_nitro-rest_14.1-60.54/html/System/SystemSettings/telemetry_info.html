telemetry_info

Configuration for Telemetry Information resource.

Read/write properties

enable_security_advisory<Boolean>

Enable Security Advisory.

id<String>

Id is system generated key for all the cloud profiles.

enable_basic_telemetry<Boolean>

Enable Basic Telemetry.

enable_optional_telemetry<Boolean>

Enable Optional Telemetry.

telemetry_mode<String>

Telemetry Mode (MANUAL or AUTOMATED).
Minimum length = 1
Maximum length = 255

host_id<String>

ADM Host id.
Minimum length = 1
Maximum length = 128

package_version<String>

Telemetry Package version.
Minimum length = 1
Maximum length = 128

Read only properties

manual_mode_selection_date<Double>

Date Manual mode was selected (EPOCH).
