authorized_scope

Configuration for Authorized Scope resource.

Read/write properties

group_id<String>

Id of the group.

managed_device_id<String>

managed device id.

formation_device_id<String>

This property is deprecated<br>formation device id.

id<String>

Auto generated ID..

managed_device_id_array<String[]>

Array for managed device.

auth_application_list<authorized_application[]>

Applications.
