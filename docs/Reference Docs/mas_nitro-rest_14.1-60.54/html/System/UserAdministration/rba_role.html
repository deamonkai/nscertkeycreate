rba_role

Configuration for RBA Role resource.

Read/write properties

tenant_id<String>

Tenant ID.
Minimum length = 1
Maximum length = 128

id<String>

Id is system generated key for all the RBA roles.

name<String>

Role Name.
Minimum length = 1
Maximum length = 128

description<String>

Description of Role.
Minimum length = 1
Maximum length = 1024

policies<String[]>

Access policies attached to this role..

Read only properties

groups<String[]>

Groups to which this role is assigned.
