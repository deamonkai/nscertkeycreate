rba_operation_client_data

Configuration for This resource is used to display the feature wise permissions tree during access policy creation in GUI. resource.

Read/write properties

id<String>

Id is system generated key.

Read only properties

parent_id<String>

Parent id for UI tree node.

operation_name<String>

This describes the operation name under a particular tree node.

dependent_resources<String>

This describes the comma separated resources, permission for which is dependent on this resource type.

serial_id<Integer>

This is used to display the permissions tree in an order manner..

display_name<String>

This describes the operation display name under a particular tree node.

type<String>

This describes the operation type for a particular tree node.

resource_type<String>

This describes the resource for a particular tree node.

parent_name<String>

Parent name for UI tree node.
