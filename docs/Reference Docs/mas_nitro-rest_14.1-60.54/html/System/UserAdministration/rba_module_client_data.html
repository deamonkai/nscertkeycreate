rba_module_client_data

Configuration for This resource is used to display the feature wise permissions tree during access policy creation in GUI. resource.

Read/write properties

id<String>

Id is system generated key.

Read only properties

parent_id<String>

Parent id for UI tree node.

child_node<String>

This is used if tree node is child node of another tree node in permission tree..

ui_component<Boolean>

This is used to mention if its UI component.

display_name<String>

module display name.

serial_id<Integer>

This is used to display the permissions tree in an order manner..

operation<rba_operation_client_data[]>

This is used to display operations of particular node in permissions tree..

submodules<rba_submodule_client_data[]>

This is used to display child nodes under a particular feature tree node in permission tree..

parent_name<String>

Parent name for UI tree node.

name<String>

module name.
