unified_appsec_profile

Configuration for unified_appsec_profile resource.

Read/write properties

profile_name<String>

Profile name.
Maximum length = 128

last_modified<String>

Last Modified time.

id<String>

Id is system generated key for protection.

app_devices<app_devices_info[]>

Mapping ADC IPs to Apps.

is_default<Boolean>

Flag to check Default Application.

parent_template<String>

Template name used to create this profile.

app_protections<protection_info[]>

protection info.

monitor_mode_toggle<Boolean>

Monitor mode toggle.

configpack_id<String>

Stylebook Config Pack ID.
Maximum length = 128

appsec_verbose_log<String>

appsec verbose log.

enable_analytics<Boolean>

Enable Analytics as part of Unified Appsec Config..

no_of_protections<String>

Number of Protections.

release<String>

NS Release.
