appsec_profile_config

Configuration for Appsec Profile config resource.

Read/write properties

ato_config<account_takeover_config[]>

Mapping ATO settings to app devices of current profile.

enabled<Boolean>

Flag to check if the profile is enabled or not..

app_devices<app_devices_info[]>

Mapping NetScaler IPs to Apps.

profile_name<String>

Profile Name used for naming a model sensitivity config.
Maximum length = 128

config_mode<Integer>

Config modes can be Model Sensitivity (0) .

model_settings<model_settings_info[]>

Appsec Model Settings info.

no_of_models<String>

Number of models.

no_of_apps<String>

Number of Applications configured.

is_default<Boolean>

Flag to check Default Application.

Read only properties

id<String>

Id is system generated key..

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
