vpn_gateway_info

Configuration for Af report for VPN Gateway resource.

Read/write properties

active_session<Double>

Number of active sessions launched by user.

gateway_ipaddress<String>

Gateway IPAddress.
Minimum length = 1
Maximum length = 128

gateway_fqdn<String>

FQDN of Gateway.
Minimum length = 1
Maximum length = 128

rpt_sample_time<Double>

Report Sample time..

device_ip_address<String>

Device IP Address..
Minimum length = 1
Maximum length = 64

total_bytes<Double>

Total bytes in sampled timeframe used by all users..

__count<Double>

count..

bandwidth<Double>

Avg Bandwidth..

id<String>

Id.
Minimum length = 1
Maximum length = 64

session_count<Double>

Number of sessions launched by user.

gateway_name<String>

Name of Gateway.
Minimum length = 1
Maximum length = 128
