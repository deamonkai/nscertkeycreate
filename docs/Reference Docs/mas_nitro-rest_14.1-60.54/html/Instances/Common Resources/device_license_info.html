device_license_info

Configuration for Device License Info resource.

Read/write properties

instance_state<String>

Managed device instance state.

details<String>

Details about license state.

flexed_instance_type<String>

Flexed instance type and count, like Multi Tenant:1,Single Tenant:6,Bare Metal:1,CPX:1.

is_las_license<Boolean>

is device LAS Licensed.

last_updated_time<Double>

Last Updated Time.

instance_mode<String>

instance mode of device.

Read only properties

license_pool<String>

License Pool.

allocated_advanced_bw<Integer>

Running advanced license capacity of VM.

hours_to_expiry<Integer>

No of Hours after which grace license expires.

allocated_standard_bw<Integer>

Running standard license capacity of VM.

current_throughput<String>

Current Througput of device.

configured_on_vm<Integer>

Configured license capacity of VM.

allocated<Integer>

Allocate license on License server.

managed_device_id<String>

Device id managaed device.

allocated_instance<Integer>

Running instance license capacity of VM.

license_state<String>

State of device, UP only if device accessible.

name<String>

Name of managed device.

grace<Boolean>

True if this device is managed by NMX.

allocated_on_vm<Integer>

Running license capacity of VM.

allocated_premium_bw<Integer>

Running premium license capacity of VM.

ip_address<String>

IP Address for this managed device.

type<String>

Type of device, (Xen | NS).
