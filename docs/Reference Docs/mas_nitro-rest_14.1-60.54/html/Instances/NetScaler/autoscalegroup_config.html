autoscalegroup_config

Configuration for Auto-Scale Group configuration/stylebook parameters resource.

Read/write properties

probe_protocol<String>

Probe Protocol.
Minimum length = 1
Maximum length = 64

access_type<Integer>

Access Type (0.External(public)/1.Internal(private)/2.None).
Maximum value =

ip_address<String>

IP of VIP.
Minimum length = 1
Maximum length = 64

backend_port<Integer>

Backend Port.
Minimum value = 1025
Maximum value =

backend_probe_port<Integer>

Backend Probe Port.
Minimum value = 11025
Maximum value =

activity_id<String>

Most recent activity_id of this autoscalegroup config.
Maximum length = 128

version<String>

Stylebook version.
Minimum length = 1
Maximum length = 16

is_https_redirect<Boolean>

Enabled for https redirect.

autoscalegroup_id<String>

ID of autoscalegroup.

stylebook_name<String>

Stylebook Name.
Minimum length = 1
Maximum length = 64

url<String>

Stylebook URL which has namespace, version and stylebook_name.
Minimum length = 1
Maximum length = 256

frontend_ip<String>

The frontend IP of Azure ALB applications.
Minimum length = 1
Maximum length = 64

last_modified_time<Double>

last modified time in seconds.

generated_domain_name<String>

Name of generated domain.
Minimum length = 1
Maximum length = 40

domain_name<String>

Name of domain.
Minimum length = 1
Maximum length = 256

probe_port<Integer>

Probe Port.
Maximum value =

redirect_port<Integer>

redirect_port.
Maximum value =

frontend_name<String>

The frontend name of application.
Minimum length = 1
Maximum length = 256

app_name<String>

Name of application.
Minimum length = 1
Maximum length = 256

id<String>

Id is system generated key for all the auto-scale group config.

virtual_port<Integer>

Virtual Port.
Maximum value =

backend_http_redirect_port<Integer>

Backend HTTP Redirect Port.
Minimum value = 21025
Maximum value =

service_type<String>

Service Type.
Minimum length = 1
Maximum length = 64

zone_name<String>

Name of zone.
Minimum length = 1
Maximum length = 128

ipset_name<String>

Name of IPSet.

is_endpoint<Boolean>

Enabled for End Point Application.

configpack_id<String>

ID of deployed configpack on autoscalegroup.
Minimum length = 1
Maximum length = 64

is_migration<Boolean>

Enabled for configpack migration.

probe_enabled<Boolean>

This flag is set to true if the NetScaler version (13.0.76.0 and above) has support for probe port and protocol.

stylebook_params<String>

Inlcudes all the Stylebook parameters.

autoscalegroup_name<String>

Name of autoscalegroup.

stylebook_namespace<String>

Stylebook Namespace.

force_delete<Boolean>

Option to delete the configpack forcefully.
