ns_cluster

Configuration for NetScaler Cluster resource.

Read/write properties

node_group<String>

Node group.

clusterid<Integer>

Cluster Id in range of 1 and 16.
Minimum value = 1
Maximum value =

clip<String>

Cluster IPAddress.
Minimum length = 1
Maximum length = 64

hostRouteEnabled<Boolean>

Is hostRoute property Enabled true/false.

state<String>

Node State. Can be ACTIVE, SPARE, PASSIVE.
Minimum length = 1
Maximum length = 32

snip<String>

Snip.
Minimum length = 1
Maximum length = 64

backplane<String>

Backplane Interface.

password<String>

Password of the Configuration Coordinator node(Cluster IP).
Minimum length = 1
Maximum length = 128

ipaddress<String>

NetScaler ipaddress which is going to participate in cluster.
Minimum length = 1
Maximum length = 64

preemption<Boolean>

Preemption.

nodeid<Integer>

Node Id in range of 0 and 31.
Maximum value =

scheduleId<String>

scheduleId is used to refer maintenaince schedule task.

process_local<Boolean>

Process Local.

iscco<Boolean>

Is Cluster coordinator.

profile_name<String>

Profile Name.
Maximum length = 128

inc_mode<Boolean>

Inc Mode.

nodegroup_list<String[]>

Available Node group list on this cluster.

add_nodegroup<Boolean>

Add Node group in cluster instance if this property is true.

clnodes<String>

List of Nodes part of Cluster.

Read only properties

rx<Double>

In Throughput of Cluster Instance in Mbps.

mgmt_cpu_usage<Double>

Management CPU Usage (%) of Cluster Instance.

clnumnodes<Integer>

Number of nodes in cluster.

operationalstate<String>

Cluster Operational State.

tx<Double>

Out Throughput of Cluster Instance in Mbps.

clcurstatus<String>

Current State of the cluster.

http_req<Double>

HTTP Requests/second on Cluster Instance.

clviewleader<String>

Cluster CCO NSIP.

health<String>

Node Health State.

act_id<String>

Activity Id.

cpu_usage<Double>

CPU Usage (%) of Cluster Instance.

memory_usage<Double>

Memory Usage (%) of Cluster Instance.
