ns_upgrade

Configuration for NetScaler Upgrade resource.

Read/write properties

mode<Integer>

Provide 0 for uploading and installing the image, 1 for only uploading the image, 2 for only installing the image and 3 for upgrade pre-validation.
Maximum value =

doc_file<String>

Documentation File Name.
Minimum length = 1
Maximum length = 128

is_pre_post_upgrade_script_same<Boolean>

True, if pre and post upgrade post failover custom scripts are same.

cleanup_unsupported_policy<Boolean>

True, if user selected to delete unsupported policies after upgrade..

issu_migration_timeout<Integer>

ISSU migration timeout value in minutes.
Minimum value = 10
Maximum value =

slack_profile<String>

Slack profile.

scheduleId<String>

scheduleId is used to refer maintenaince schedule task.

is_issu_enabled<Boolean>

True, if ISSU is enabled.

ns_ip_address_arr<String[]>

List of NetScaler IP Address.

second_failover_enabled<Boolean>

True, if second failover for HA upgrade is enabled.

device_groups<String[]>

Device Group Array on which for which job is run.

ha_node2_devices<String[]>

List of HA NetScaler IP Address upgrade in second phase.

adc_cloud_image_id<String>

NetScaler Cloud Image ID to which AutoScaleGroup(s) is/are to be upgraded.
Minimum length = 1
Maximum length = 128

default_configdiff_template<String[]>

List of default upgrade templates.

post_upgrade_script<String>

Name of post upgrade script file.
Maximum length = 128

is_pre_post_upgrade_pre_failover_script_same<Boolean>

True, if pre and post upgrade pre failover custom scripts are same.

is_post_upgrade_script_selected<Boolean>

True, if post upgrade custom script is selected.

adc_cloud_image_version<String>

NetScaler Cloud Image Version to which AutoScaleGroup(s) is/are to be upgraded.
Minimum length = 1
Maximum length = 128

is_pre_upgrade_script_selected<Boolean>

True, if pre upgrade custom script is selected.

do_cleanup<Boolean>

True, if user wants to clean the image files once upgrade is completed successfully.

mail_profiles<String>

Comma separated list of Mail profiles.

scheduleTimesEpoch<String>

Schedule time epoch (string representation of 11 digit numbers)..

delay_ha_failover<Boolean>

True, if detal two phase HA failover.

name<String>

Name of upgrade.
Maximum length = 128

is_post_upgrade_pre_failover_script_selected<Boolean>

True, if post upgrade pre failover custom script is selected.

disable_build_image_upload_to_primary<Boolean>

True, if disable build image upload to HA primary device.

do_backup<Boolean>

True, if user wants to take backup before ns upgrade.

post_upgrade_pre_failover_script<String>

Name of post upgrade pre failover script file.
Maximum length = 128

image_name<String>

image_name.
Maximum length = 128

saveconfig_enabled<Boolean>

True, if save config before upgrade.

scheduleTimesEpoch2<String>

Schedule time epoch for ha primary node (string representation of 11 digit numbers)..

type<String>

Device type for NS device family.
Maximum length = 64

ha_two_phase_upgrade<Boolean>

True, if user wants to upgarde ha in two phases or steps.

id<String>

Id is maintenance job id.

pre_upgrade_script<String>

Name of pre upgrade script file.
Maximum length = 128

is_skip_upload_image<Boolean>

True, if skip upload build image.

is_remote_download<Boolean>

True if upgrade is remote download instead of direct upload from NetScaler Console.

Read only properties

act_id<String>

Activity Id.

tasklog_id<String>

Task Log Id of the Config Job.
