autoscalegroup_profile_map

Configuration for Auto-Scale Group provision details for each availability zone resource.

Read/write properties

retry_err_type<String>

Enable/Disable provisioning retry.
Maximum length = 50

provision_request_id<String>

The request id generated for provisoning of cluster from provisioning service.
Maximum length = 128

provision_job_id<String>

The job id generated for provisoning of failed node from provisioning service.
Maximum length = 128

id<String>

Id is system generated key for all the profiles.

bandwidth_limit<Integer>

Bandwidth limit per ADC.

parent_name<String>

Parent name of autoscale group.

error_message<String>

Provision error message.
Maximum length = 255

is_availability_set<Boolean>

Azure: If this profile belongs to availability set.

node_count<Integer>

Number of nodes to be provisioned in the cluster.
Minimum value = 1
Maximum value =

availability_zone<String>

availability_zone of autoscalegroup.
Minimum length = 1
Maximum length = 128

az_version<String>

Version of availabilty_zone/cluster. Auto generated..

az_type<Integer>

Type of availabilty_zone/cluster (1.ORIGINAL, 2.BACKUP).
Minimum value = 1
Maximum value =

is_active_node_prov<Boolean>

This is to refer if the provisioning inprogress/failed/success is on active/passive node.

activity_id<String>

Most recent activity_id of this availability_zone/cluster.

node_id<String>

The managed_device id which has reference to the node provisioned or to be deprovisioned.
Maximum length = 128

azure_resource_group_name<String>

Resource group name for Azure.
Maximum length = 128

provision_profile_id<String>

ID of provision profile.
Minimum length = 1

cluster_id<String>

NetScaler Cluster ID for the availability zone.
Maximum length = 128

retry_count<Integer>

current auto retry count for a retryable error in a cluster..
Maximum value =

parent_id<String>

Parent Id of autoscalegroup.

error_type<Integer>

For mock test: simulates response type. 0-success,1-failed,2-validation failed with no delay.

autoscalegroup_name<String>

Auto-Scale Group Name.

delay<Integer>

For mock test: Delay in seconds introduced to get the response..
Maximum value =

Read only properties

cluster_ip_address<String>

NetScaler Cluster IP Address.
