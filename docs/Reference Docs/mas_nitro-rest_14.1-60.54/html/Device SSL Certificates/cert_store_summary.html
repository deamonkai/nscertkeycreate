cert_store_summary

Configuration for SSL certificate store summary on NetScaler Console resource.

Read/write properties

cert_usage<property_map[]>

Certificate by its usage.

cert_algorithm<property_map[]>

Certificates by signature algorithm.

cert_expiry<property_map[]>

Certificates by expiry.

cert_sign<property_map[]>

Certificates by signing(self/ca).

cert_key<property_map[]>

Certificates by key strength.

cert_bound_apps<property_map[]>

Applications bound to each certificate.

cert_issuer<property_map[]>

Certificate by issuer.

Read only properties

total_certs<Integer>

Total number of certificates.
