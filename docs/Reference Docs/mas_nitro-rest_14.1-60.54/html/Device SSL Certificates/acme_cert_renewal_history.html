acme_cert_renewal_history

Configuration for ACME Certificate Issuance and Renewal History resource.

Read/write properties

renewed_by_ca_config<String>

Name of the CA configuration which is used to renew this certificate. This is the same as the name of the CA configuration in acme_ca_config table..

cert_id<String>

Id of the certificate which is renewed. This is the same as the ID of the certificate in cert store..

name<String>

Name or alias which identifies certificate and key pair.

id<String>

Id is system generated key for all certificate renewal history entries..

zero_touch_certificate<Boolean>

Set to true, if the certificate is uploaded for zero touch ssl certificate repository purpose..

username<String>

User who performed the renewal/issuance operation. This is applicable when the certificate is renewed manually. For auto renewal, it will be system..

last_renewed_at<Double>

Last renewal time for this certificate in seconds. This is applicable when the certificate is enabled for auto renewal..

renew_mode<String>

Renew mode for the certificate. It can be MANUAL or AUTO. If it is set to AUTO, then the certificate will be auto renewed based on the CA config associated with this certificate..

domain<String>

Domain name from the subject of the certificate.

renewal_status<String>

Last renewal status (SUCCESS/FAILED) for this certificate. This is applicable when the certificate is enabled for auto renewal..

action<String>

Action to get the summary of the certificate renewal issuance/history.

duration<String>

Duration to filter the certificate renewal issuance/history. It can be 'last_24_hours', 'last_7_days', 'last_30_days', 'last_90_days', 'last_365_days' or 'all'..

status_summary_map<property_map[]>

Array of tag_name and tag_value pair assocaited with an entity.

Read only properties

log_message<String>

Log message for the last renewal operation. This is applicable when the certificate is renewed manually/anutomatically..
