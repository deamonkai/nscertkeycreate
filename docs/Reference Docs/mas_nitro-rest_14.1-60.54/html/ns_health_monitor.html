ns_health_monitor

Configuration for Health Monitor for ADC resource.

Read/write properties

monitor_state<String>

State of monitor possible values UP, DOWN, UNKNOWN, BUSY, OUT OF SERVICE, GOING OUT OF SERVICE, DOWN WHEN GOING OUT OF SERVICE, NS_EMPTY_STR, Unknown, DISABLED.

monstatparam2<Integer>

Second parameter for use with message code.

monitortotalfailedprobes<String>

Total number of failed probes.

monitorcurrentfailedprobes<String>

Total number of currently failed probes.

monstatparam1<Integer>

First parameter for use with message code.

monitortotalprobes<String>

Total number of probes sent to monitor this service..

monstatcode<Integer>

The code indicating the monitor response.

monstatparam3<Integer>

Third parameter for use with message code.

svc_grp_name<String>

The NetScaler Console.

ns_ip_address<String>

NetScaler IP Address.
Minimum length = 1
Maximum length = 64

svc_name<String>

service name monitor associated with.

monitor_name<String>

Name of the monitor.

lastresponse<String>

The string form of monstatcode.

responsetime<String>

Response time of this monitor.

Read only properties

partition_name<String>

Partition Name.

poll_time<Integer>

Last Polling Time.
