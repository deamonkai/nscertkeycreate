ns_waf_relaxation

Configuration for Reference object to pass for relaxing a WAF Learning engine learnt rule resource.

Read/write properties

min_value<Integer>

minimum length of the field to be relaxed.

http_req_url<String>

HTTP Request URL to be relaxed.
Maximum length = 2000

value_expr<String>

Value expression to be relaxed.
Maximum length = 512

mode<Integer>

mode to be deployed as open/closed.

max_value<Integer>

Maximum length of the filed to be relaxed.

violation_id<Integer>

Violation Category.

comment<String>

Comment to be sent as payload in relaxation rule nitro.
Maximum length = 255

value_type<String>

Value type to be relaxed.
Maximum length = 512

status<Boolean>

Status is true when the rule is enabled and false when the rule is disabled.

field_type<String>

Field type to be relaxed.
Maximum length = 512

resourceid<String>

Resourceid generated for each relaxation rule..
Maximum length = 128

violation_location<Integer>

Violation Location.

adm_learn_profile_name<String>

adm_learn_profile_name.
Minimum length = 1
Maximum length = 32

si_device_ip_address<String>

NetScaler IP Address.
Minimum length = 1
Maximum length = 64

field_name<String>

Field name to be relaxed.
Maximum length = 512

profile_name<String>

profile_name.
Minimum length = 1
Maximum length = 32
