ns_servicegroup

Configuration for NetScaler Service Group resource.

Read/write properties

id<String>

Id is system generated key.

svc_grp_mem_ip_address<String>

Service Group Member IP Address.
Minimum length = 1
Maximum length = 64

svc_grp_mem_port<Integer>

Service Group Member Port.
Minimum value = 1
Maximum value =

hostname<String>

Hostname of the managed device.
Minimum length = 1
Maximum length = 256

ns_ip_address<String>

NetScaler IP Address.
Minimum length = 1
Maximum length = 64

svc_grp_name<String>

Service Group Name.
Minimum length = 1
Maximum length = 100

appflowlog<String>

AppFlow is enabled at servicegroup of LBVServer.

lb_id<String>

LB ID.

graceful<Boolean>

Graceful shutdown of Service..

vsvr_name<String>

Vserver Name.
Minimum length = 1
Maximum length = 100

svr_name<String>

Server Member Name.
Minimum length = 1
Maximum length = 100

svr_id<String>

Server ID.

save_config<Boolean>

Save configuration after enable/disable.

delay<Double>

Time, in seconds, allocated to the NetScaler appliance for a graceful shutdown of the service.

Read only properties

partition_name<String>

Partition Name.

svc_grp_effective_state<String>

Service Group Effective State.

svc_grp_mem_type<String>

Service Group Member Protocol Type.

display_name<String>

Display Name.

svc_grp_mem_state<String>

Service Group Member State.

comment<String>

Comment.

svrstate<String>

Service State.

Operations

NOTE: In request payload, Mandatory parameters are marked red and bold.

The following parameters can be used in the nitro request :

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
