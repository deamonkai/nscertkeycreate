lb_export_report

Configuration for Export or Schedule NetScaler Entities report resource.

Read/write properties

mail_profile<String>

Mail Profile.

slack_profile<String>

Slack Profile Name.

recurrence_type<String>

Tenant recurrence type.
Maximum length = 255

enable_auth<Boolean>

Enable Auth Report.

enable_lb<Boolean>

Enable LB Report.

is_enabled<Boolean>

Enable Scheduled Report (default is enabled).

enable_gslb<Boolean>

Enable GSLB Report.

enable_sslvpn<Boolean>

Enable SSLVPN Report.

export_now<Boolean>

Load Balancing Export Now.

recurrence_options<String>

Comma separated recurrence options of job that is scheduled.

file_name<String>

Export Report File name..

recurrence_times_epoch<String>

Schedule time epoch (string representation of 11 digit numbers)..

enable_cr<Boolean>

Enable CR Report.

enable_cs<Boolean>

Enable CS Report.

Operations

onerror   <String_value>

Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.
Default value: EXIT
Possible values = EXIT, CONTINUE
