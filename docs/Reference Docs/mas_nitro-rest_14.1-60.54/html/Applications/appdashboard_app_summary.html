appdashboard_app_summary

Configuration for Provides the app dashboard full summary resource.

Read/write properties

total_bytes<Double>

Session total bytes.

appcategory<String>

App Category.
Maximum length = 255

health_score_str<String>

App Health Score.

error_percentage<Double>

Number of 5xx response received wrt total requests.

appname<String>

AppName.
Maximum length = 255

srvr_connections<Double>

Current Server Connections.

response_time<Double>

total time taken by the app to send response to client.

clnt_connections<Double>

Current Client Connections.

total_requests<Double>

Total Requests.

curr_state<String>

Current State.
Maximum length = 50

throughput<Double>

Throughput ..

apps_affected<String[]>

All parent application names separated by comma that are affected by this referenced application.

ip_address<String>

Field to store the ip address as it is along with any extension.
Maximum length = 512

vsvr_ip_address<String>

Vserver IP address.

health_score<Double>

App Health Score.

pkt_sent_cur<Double>

Total Pkts Sent.

issues_detected<String>

Array of all app issues affecting health score. Each array element tells issueId, penalty and time when it occured.

hostname<String>

NetScaler Hostname.

ssl_rating<String>

A+ / non-A+ / NA.
Maximum length = 50

vslbhealth<Integer>

Vserver Health.

pkt_recvd_cur<Double>

Total Pkts Received.

tags<String>

Tags associated with apps.

Read only properties

application_class<String>

Application class.
