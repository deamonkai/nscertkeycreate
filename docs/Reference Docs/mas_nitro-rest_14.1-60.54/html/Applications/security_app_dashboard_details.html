Use this parameter to set the onerror status for nitro request. Applicable only for bulk requests.

Possible values = EXIT, CONTINUE

get (all)

get (all)

Query-parameters:

order_by

Use this query-parameter to get the result that is ordered by the given property.

Possible Values:total_attacks, app_threat_index

Use this query-parameter to get the result that is grouped by according to the given property.

Possible Values:violation_type, rpt_sample_time
