appscore_factor

Configuration for Factors contributing in appscore calculation resource.

Read/write properties

contribute<Boolean>

Flag to check if factor is going to contribute in appscore..

factor_name<String>

Factor Name..

factor_id<Integer>

Factor If..

Read only properties

id<String>

Id.
